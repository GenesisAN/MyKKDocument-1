# 常见问题

::: tip 关于无网络访问
网站使用PWA(Progressive Web Apps)技术，在无网络，服务器停机的情况下只要你曾经打开过对应网页，那么那些网页依旧可正常被访问，右上角的搜索功能也能正常使用
:::

::: tip 关于新内容刷新
在网页右下角出现新内容提示时，记得点击刷新，那是基于PWA实现的本地缓存刷新，让其内容与服务器的最新内容保持一致
:::

网站将会从三种身份，来提供一个FAQ指南，请点击你感兴趣的目录快速跳转
